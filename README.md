## 🚀 Quick Start

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- The birthday person's photo as `eshita.jpg`

### Setup Instructions

1. **Place the Photo**
   - Ensure `eshita.jpg` is in the same directory as `index.html`
   - The photo should ideally be a portrait or close-up shot
   - Recommended size: 800x800px or larger for best quality

2. **Open in Browser**
   - Double-click `index.html` or
   - Right-click `index.html` → Open with → Your preferred browser

3. **Local Server (Optional but Recommended)**
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```
   Then visit `http://localhost:8000` in your browser.

## 🆘 Troubleshooting

### Photo not showing?
- Ensure `eshita.jpg` is in the same folder as `index.html`
- Check file name spelling (case-sensitive)
- Try JPG or PNG format
- Verify file is not corrupted

### Animations not smooth?
- Close unnecessary browser tabs
- Update browser to latest version
- Disable browser extensions
- Try a different browser

### Confetti not working?
- Ensure JavaScript is enabled in browser
- Check browser console for errors (F12)
- Try the celebration button again
- Refresh the page

### Mobile layout issues?
- Refresh page after rotating device
- Check viewport meta tag in HTML
- Update browser to latest version
